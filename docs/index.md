# KBox
KBox is a very small but useful utility library for Kotlin (JVM, Android and JS).

# Installation

KBox is published to maven central.

```
repositories { mavenCentral() }
dependencies {
    implementation("ch.tutteli.kbox:kbox:2.2.0")
}
```
